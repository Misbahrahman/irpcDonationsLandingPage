## Image Optimization Tips

- **Format**: Convert to WebP for better performance
- **Size**: Keep under 500KB per image
- **Dimensions**: Max 1920px width, maintain aspect ratio
- **Compression**: Use tools like TinyPNG, Squoosh, or ImageOptim
- **Quality**: 75-85% JPEG quality is usually sufficient
- **Content**: Show authentic, emotional moments from IRPC activities

## Tools for Optimization

- [Squoosh](https://squoosh.app/) - Online image compression
- [TinyPNG](https://tinypng.com/) - PNG/JPG compression
- [ImageOptim](https://imageoptim.com/) - Mac app for image optimization
- [GIMP](https://www.gimp.org/) - Free image editing
