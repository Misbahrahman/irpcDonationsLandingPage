# Banner Photos

Place your banner photos here with these exact names:

- `banner1.jpg` - Your best/most impactful photo
- `banner2.jpg` - Second photo
- `banner3.jpg` - Third photo
- `banner4.jpg` - Fourth photo
- `banner5.jpg` - Fifth photo

## Image Specifications

- **Format:** JPG, JPEG, PNG, or WebP
- **Recommended Size:** 1920 x 1080 pixels
- **File Size:** < 500KB each (optimized)
- **Aspect Ratio:** 16:9 (landscape)

## Download Your Photos

From your IRPC project repository:
https://github.com/Misbahrahman/project_irpc/tree/main/public/images/BannerPhotos

Then rename and place them here as:
- banner1.jpg
- banner2.jpg
- banner3.jpg
- banner4.jpg
- banner5.jpg

## Quick Test

After adding photos, run:
```bash
npm run dev
```

Visit http://localhost:3000 to see the carousel in action!
